# Решения задач по олимпиадному программированию
Репозиторий содержит набор хуевых решений задач и заготовок для тренировок и этапов соревнований.

Всегда найдётся тот, кто решит, что любое решение — списанное, и тот, кто посчитает, что на следующем этапе автор наберёт ноль баллов. Это не очень важно. Репозиторий хранит решения и идеи ради обучения и прогресса, а не ради спекуляций. Хотя...

## О проекте
Коллекция файлов с решениями на Python для разных этапов и наборов олимпиадных задач. Цель — хранить компактные, понятные реализации задач для повторения и подготовки.
